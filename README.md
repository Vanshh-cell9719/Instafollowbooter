# InstaPromoBoost Telegram Bot

InstaPromoBoost is a Telegram bot that helps users grow their Instagram accounts by completing simple follow tasks.

## Features
- Instagram follow tasks
- Reward system
- User profile with points
- Telegram command menu
- SEO optimized bot

## Commands
/start - Start the bot  
/earn - Complete Instagram follow tasks  
/profile - View your stats  
/menu - Main menu  
/help - How to use the bot  

## How it works
Users follow Instagram pages to unlock rewards and grow their Instagram reach.

## Deployment
This bot runs on Render cloud hosting using Python and python-telegram-bot.
